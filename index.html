어울림 성격 유형 검사

(Awoolim Type Test, ATT)

어울림: 개인이 사회·집단 속에서 어떤 역할과 방식으로 관계를 형성하고 흐름을 조율하는지 보여주는 구조

어울림 유형 소개

🔹 에너지 방향: 리드(L) ↔ 플로우(F)

리드(L, Lead / 조율자형)

팀이나 모임에서 주도적으로 방향을 잡고 결정을 이끌어가는 스타일입니다. 순간적인 결단력과 조율 능력으로 상황을 관리하며, 다른 사람들이 자연스럽게 따르게 만드는 힘이 있습니다.

플로우(F, Flow / 따르는 흐름형)

주도보다 주변 상황과 팀의 흐름을 관찰하며 따라가는 스타일입니다. 상황과 사람의 감정을 읽고 적응하며, 안정적이고 균형 있는 역할을 수행합니다.

🔹 정보·표현 방식: 표현(E) ↔ 감응(R)

표현(E, Expression / 언어·행동 중심)

말과 행동을 통해 자신의 생각과 감정을 표현하며, 주변 사람들과 직접적으로 상호작용하는 것을 선호합니다. 눈에 보이는 행동과 즉각적 피드백으로 영향력을 발휘합니다.

감응(R, Response / 맥락·관계 중심)

상황의 맥락, 사람들의 관계, 비언어적 신호를 민감하게 읽는 스타일입니다. 말보다는 관찰과 이해를 통해 영향력을 발휘하며, 팀과 조직의 흐름을 조율합니다.

🔹 복합형 전용 알파벳: P / C

P (Performance, 표현 편향)

리드와 표현 성향이 강하게 섞인 복합형에서, 순간적 몰입과 퍼포먼스로 주변을 이끄는 능력이 두드러집니다.

C (Contextual, 감응 편향)

리드와 감응 성향이 섞인 복합형에서, 장기적 흐름과 관계 조율 능력이 두드러집니다.

어울림 검사는 사회적 관계 속에서 내가 어떤 역할을 하는지 알려주는 검사입니다.
총 20문항, 3분 내외 소요됩니다.
